# SportsScraper
### Description:
Web scraper for columbus sports information. Sends email to desired
 recipient(s) of most recent articles from a number of sites
  currently relating to:
* OSU football
* Blue Jackets Hockey
* Generic OSU sports

### Websites Currently featured:
* Eleven Warriors
* Blue Jackets on NHL.com
